jQuery Ajax配置细节

总结jquery的ajax基本配置信息.同时对比给出原生ajax的基础兼容写法

options:可选。AJAX 请求设置。所有选项都是可选的。
async:是否异步加载
beforeSend(XHR):发送请求前可修改 XMLHttpRequest 对象的函数，如添加自定义 HTTP 头。返回false可取消本次请求.
complete(XHR, TS):请求完成后回调函数 (请求成功或失败之后均调用)。
data:发送到服务器的数据。将自动转换为请求字符串格式。GET 请求中将附加在 URL 后。
dataType:预期服务器返回的数据类型。
error:请求失败调用函数.
success:请求成功调用函数.
url:发送请求的地址

原生Ajax

readyState有五种状态：
　　0 (未初始化)： (XMLHttpRequest)对象已经创建，但还没有调用open()方法；
　　1 (载入)：已经调用open() 方法，但尚未发送请求；
　　2 (载入完成)： 请求已经发送完成；
　　3 (交互)：可以接收到部分响应数据；
　　4 (完成)：已经接收到了全部数据，并且连接已经关闭。

status是一种辅状态判断，status是服务器方的状态判断：
　　100——客户必须继续发出请求
　　101——客户要求服务器根据请求转换HTTP协议版本
　　200——成功
　　201——提示知道新文件的URL
　　300——请求的资源可在多处得到
　　301——删除请求数据
　　404——没有发现文件、查询或URl
　　500——服务器产生内部错误
