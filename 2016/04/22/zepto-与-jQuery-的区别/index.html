zepto 与 jQuery 的区别

zepto总的来说与jquery基本没啥区别了,但是作为移动端的简化的jquery,zepto自然是与jquery有所不同的.

注意:zepto官方下载的内容不完整,缺少很多重要的模块,在zepto build里面生成自己需要的js文件

一.选择器区别
1 :text :checkbox :first 等等在 jQuery 里面很常用的选择器，Zepto 不支持
Zepto 是直接通过浏览器提供的 document.querySelectorAll 接口.添加selector模块后能够支持 部分 的 jQuery 选择器扩展:
:visible :hidden
:selected :checked
:parent
:first :last :eq
:contains :has

2 Zepto的选择器表达式:[name=value]中value 必须用双引号”or单引号’括起来

3 zepto在操作dom的selected和checked属性时尽量使用prop方法,因为prop方法会优先于attr

4 Zepto 是根据标准浏览器写的，所以对于节点尺寸的方法只提供 width() 和 height()，省去了 innerWidth(),innerHeight(),outerWidth(),outerHeight()
具体宽高由盒模型决定.(隐藏元素无法获取宽高)

5 zepto 的offset()方法
Zepto.js: 返回 top 、 left 、 width 、 height
jQuery: 返回 width 、 height
