Git基础指令总结

项目中必定会使用git,现在自己对git的一些基础操作命令还有些不适应,经常需要去翻查具体的指令,所以这篇文章用来记录常用的命令给自己作为查询

常用指令

1.git remote -v //查看远程分支的git路径
2.git clone git@xxxxx.git
3.git status //查看当前版本的状态
4.git add xxx //添加当前修改的文件到暂存区
5.git commit -m ‘更新的内容描述’ //提交暂存区更改
6.git rm xxx //完全删除文件
7.git rm -cached xxx //仅在暂存区删除,保存文件在当前目录,不再跟踪
8.git mv oldname newname //重命名文件
9.git checkout -b [分支名] //创建新分支,新分支来自于当前分支
10.git checkout [分支名] //切换至分支
11.git merge dev //合并dev分支到当前分支
12.git branch //产看本地分支
13.git pull origin //更新远程代码到本地分支
14.git push origin [远端分支名] //push本地分支到远端分支,如果远端没有该分支则会自动创建
15.git branch -d [分支名] //删除本地分支
