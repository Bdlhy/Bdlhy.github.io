FEkit常用命令与基础服务搭建

fekit init 注册fekit进项目文件中,把所有css,js文件放入src文件夹,然后通过主要js或者主要css文件进行集体require打包

fekit config文件设置,以下内容注意:
dependencies: 依赖的外部组件
compiler: 开发产品还是组件
export: 输出的文件(可以直接访问的)
alias: 别名,缩短路径

fekit路径: 在页面中调用绝对路径,公式如下
主机地址 + 文件夹路径 + prd + src下文件路径 + @版本随机码(方便缓存)
http://localhost/testFEkit/prd/scripts/index@VERSION.js

fekit server -c 合并文件访问,页面中只用加载出一个总文件即可
fekit server -p 81 设置端口号,可以在不同的端口运行服务可以开启多个服务

moudule.exports = XXX 暴露对象方式
exports.XXX = function(){}直接暴露方法
xxx = require(./xxxxxx/xxx.js)
xxx.XXXX()调用
