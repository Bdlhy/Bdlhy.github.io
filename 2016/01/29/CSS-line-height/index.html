CSS line-height

CSS line-height

一、line-height 适用范围

1、对于块级元素，CSS属性line-height指定了元素内部inline-boxes的最小高度。
即p div等元素每一行内容的最小高度
2、对于非替代行内元素，line-height用于计算line box的高度。
即span等元素的内容高度
3、对于替代行内元素，如button 或其他input元素，line-height没有影响。

二、line-height 取值范围

1、normal
取决于用户代理。桌面浏览器（包括火狐浏览器）使用默认值，约为1.2，这取决于元素的font-family。
理解：根据不同的浏览器，其实自带有其本身的文字行高，用normal来调用。
line-height: normal;

2、（无单位数字）（推荐！）
计算方法是用数字乘以元素的font-size属性
line-height: 3.5;
被继承后可以达到子元素自适应行高。这种方法对于文字来说更为推崇。

3、
用不同的计量单位直接定义line box的高度
line-height: 3em;
会被子元素继承，高度固定后不变，不会做相应比例的缩放

4、
相对于元素自身的字体大小。计算出的值是给定的百分比值乘以元素计算出的字体大小。
line-height: 34%;
这种情况line-height只会用百分比与对应设定的字体大小相乘。但是只是应用于对应区块。不一样的字体大小导致其他地方需要重新设定。会被元素继承
