样式重置 CSS reset

CSS reset！样式重置！

1、什么是样式重置？

对于我们浏览器来说，许多页面元素其实有它特定的样式，比如img自带1px边框，比如h1自动margin-top，又比如a的颜色是蓝色，自带下划线。
目前很多美工设置的情况下，这些样式其实是不美观的，同时对于不同浏览器存在着默认配置有差异容易引起兼容性问题。
所以大家通过进行css reset，重置这些元素的默认样式，达到统一，整洁的效果。

2、样式重置的误区

样式重置的作用只是为了避免页面中元素表现情况出现不统一的后果，但是由于很多时候没有具体分析元素的需求，导致很多没有必要的元素样式也进行了重置，这种做法有以下几点缺陷：

（1）影响css文件的大小！多余的，不简洁的样式重置带来了很多没有用处的代码，影响加载速度。

（2）样式的复写！很多重要的样式例如div，a这些我们都会去添加自己希望的样式，那么对他们进行reset就意义不大了。

（3）css渲染！reset css中，如果包含一些重要的元素，那么在一开始进行页面的渲染时是会花很大工作量的。
